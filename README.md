# embeddedCalculator
A calculator specifically to support embedded software development

NOTE: This project is vibe coded within about 1 hour. THe usability might still be improved.


The calculator is specifically made to support embedded software development. In embedded software devlopment, different number bases are used on a regular basis.
The main bases used is hexadecimal (base 16), decimal (base 10), octal (base 8) and binary (base 2). Optionally also float and double can be used.

The current number is then shown in all bases at the same time. Additionally, there is a bit field view where the bit positions are shown. Not only are the 
bit shown but fields can be defined. The calculator then shows the value in these fields. This can be used to understand the settings written to or read from
a register.
